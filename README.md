# SOULTEST - SCL-90心理健康测试网站

一个专业的心理健康测试网站，基于SCL-90症状自评量表，提供全面的心理健康评估服务。

## 项目简介

SOULTEST是一个专注于心理健康测试的在线平台，采用国际通用的SCL-90症状自评量表，帮助用户进行科学、客观的心理健康评估。网站支持移动端、平板和桌面端三端适配，所有测试数据仅存储在用户设备本地，保障隐私安全。

## 核心功能

- **SCL-90量表测试**：包含90道题目，涵盖9个维度的心理健康评估
- **5级评分系统**：从"从无"到"严重"的5级评分标准
- **详细结果分析**：提供总分、因子分、症状分布等多维度分析
- **专业建议**：根据测试结果提供个性化的心理健康建议
- **三端适配**：完美支持手机、平板和桌面电脑
- **本地数据存储**：所有测试数据仅存储在用户设备本地，不上传服务器

## 部署指南

### 本地部署

1. 克隆项目到本地：
   ```
   git clone https://github.com/yourusername/soultest.git
   ```

2. 直接在浏览器中打开index.html文件，或使用本地服务器运行：
   ```
   cd soultest
   python -m http.server 8000
   ```
   然后在浏览器中访问 `http://localhost:8000`

### 线上部署

1. 将index.html文件上传到您的Web服务器
2. 确保服务器支持静态HTML文件的访问
3. 无需额外的后端服务，网站是纯前端实现

## 使用说明

1. 打开网站后，点击"开始测试"按钮进入测试页面
2. 认真阅读每道题目，根据您的实际情况选择相应的选项
3. 完成所有题目后，系统将自动生成测试结果和分析报告
4. 您可以查看详细的结果分析和专业建议
5. 测试结果会保存在您的浏览器本地存储中，下次访问时可以查看历史记录

## 注意事项

- 本测试仅供参考，不能替代专业的心理咨询或诊断
- 测试结果仅反映当前的心理状态，随着时间和环境的变化可能会有所不同
- 请根据您的实际情况如实回答，以获得更准确的评估结果
- 如果您的测试结果显示存在较严重的心理问题，建议及时时寻求专业心理咨询师或医生的帮助
